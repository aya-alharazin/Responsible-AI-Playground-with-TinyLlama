# 🛡️ TinyLlama Responsible AI Playground

## 📌 Project Overview

This project explores important Responsible AI concepts using the TinyLlama foundation model on Kaggle.

The notebook demonstrates how foundation models behave differently when given:
- safe prompts
- unsafe prompts
- toxic instructions
- and guarded prompts

The project focuses on understanding:
- AI safety
- toxicity generation
- prompt engineering
- foundation model behavior
- and AI guardrails.

---

# 🤖 What is TinyLlama?

TinyLlama is a lightweight open-source foundation model (FM) that can generate human-like text responses.

It is useful for:
- learning generative AI concepts
- experimenting with prompts
- understanding LLM behavior
- and studying Responsible AI principles.

The notebook also demonstrates simple rule-based guardrails that help block unsafe prompts before generation.

# 📊 Key Responsible AI Observations

The experiments show that:

- Foundation models can generate both helpful and harmful outputs.
- Smaller models may lack strong safety alignment.
- Prompt wording strongly influences AI behavior.
- AI systems require safeguards and moderation layers.
- Raw foundation models are primarily text continuation systems.
